True flase in condition:

ways of truthy

  1. asign the value of a variable True
  2. asign the value of a variable any number positive or negitive without 0 but string type '0' treated as string and it will be truthy
  3. asign the value of a variable any string without empty string (''), but
  4. empty object {} or empty array [] will be truthy

ways of falsy

  1. asign the value of a variable false but string type 'false' return true because it treated as string not boolean.
  2. number 0 will be falsy
  3. empty string '' but wihte space in empty string (' ') treated as string and it will return true
  4. undefined
  5. null

double equal sign (==) vs triple equal sign (===) in comparision:

double equal sign(==) retuns true if only values of both are equal on the other hand triple equal sign (===) returns true if values and types of both are equal.

যখন কোনো boolean value এর সাথে অন্য কোনো টাইপের ডেটাকে double equal(==) দিয়ে compare করা হয়, তখন javascript ভিতরে ভিতরে দুইটাকেই number এ convert করে, এবং সে হিসেবেই রেজাল্ট রিটার্ন করে। আর এটাকে type coercion বলে।
