Javascript Object

Object:

Object হল non-primitive ডেটা টাইপ। প্রায় সব প্রোগ্রামিং ল্যাংগুয়েজেই অবজেক্ট আছে।

We can define object thus: An unordered collection of key-value pairs. Each key-value pair is called a property.

নিচে Object এর কিছু বৈশিষ্ট দেয়া হল:

  1. Object সাধারণত const দিয়ে ডিক্লেয়ার করা হয়।
  2. Object কার্লি ব্রেস {} এর মধ্যে লেখা হয়।
  3. Object এ একটি property ও একটি value দিয়ে ডেটা স্টোর করা হয়।
  4. Object এর মধ্যে সব ধরণের ডেটা টাইপ তথা string, number, boolean, null, undefined, array, function এমনকি object ও ভ্যালু হিসেবে স্টোর করা যায়।
  5. Difference between object and array :
    • Array stores data in ordered(sequential) collection but Object stores data in unordered collection.
    • Data can be accessed in array using numerical index. for object, values are accessed calling keys.

Object Declaretion:

Object কে নিম্নোক্ত ভাবে declare করা যায়।

  1. Using an object literal. const objectName = {key:value, key:value.....}
  2. Using new keyword const objectName = new Object(key:value, key:value.....)
  3. Using Obeect.create const objectName = Object.create(key:value, key:value.....)
  4. Using class or function-- Define an object constructor. then create objects of the constructed type.

Object Properties:

Object এর কোনো একটি property এর ভ্যালু এক্সেস করতে বা নতুন ভাবে সেট করতে dot notation ও bracket notation ব্যবহার করা হয়।

  1. with dot notation -> objectName.propertyName
  2. with bracket notation -> objectName[propertyName] এক্ষেত্রে property name কে একটি স্ট্রিং হিসেবে লিখতে হবে। অর্থাৎ সিঙ্গেল বা ডাবল কোটেশনের মধ্যে লিখতে হবে।
  3. কোনো একটি ভেরিএবল এর ভ্যালু হিসেবে property এর নামকে সেট করে তারপর সেই ভেরিএবলকে bracket notation এর মধ্যে বসিয়ে দিয়েও property এর ভ্যালু এক্সেস করা যায়।

তবে propertyName একাধিক ওয়ার্ডে হয় বা _, বা এরকম কিছু স্পেশাল কী দিয়ে শুরু হলে অথবা কোনো একটি ভেরিএবল এর ভ্যালু হলে dot notation দিয়ে property এর ভ্যালু এক্সেস বা সেট করা যায় না। সেক্ষেত্রে bracket notation হল একমাত্র সমাধান।

Keys, Values, nested objects and delete

  1. কোনো একটি Object এ কতগুলো কী বা প্রপার্টি আছে তা দেখতে চাইলে Object.keys(ObjectName) returns an array with object property Name.
  2. কোনো একটি Object এর যতগুলো ভ্যালু আছে তা বের করতে চাইলে Object.values(ObjectName) returns an array with object values.
  3. nested object এর value বের করতে যতবার নেস্টেড ততবার dot notation বা bracket notation ও প্রপার্টি নেম দিতে হবে।
  4. object এর কোন প্রপার্টি ডিলিট করতে চাইলে delete objectName.propertyName দিতে হবে।

Destructuring:

  1. A tichnique for unpacking object, array, and assigning them to variables.
  2. Helps to deal with function that have a lot of parameters, default values.
  3. Is convenient to write, easy to maintain and friendly to read.
  4. Saves from writing repetitive code.

object কে Destructuring করতে হলে

Object.keys:

আমরা কোনো object এর key গুলোকে পেতে চাইলে,
const keys = Object.keys(objectName);
তাহলে এটি একটি array রিটার্ন করবে যাতে object এর key গুলো থাকবে।

একইভাবে কোনো object এর property গুলোর value গুলো একসাথে পেতে চাইলে,
const values = Object.values(objectName);
তাহলে এটি একটি array রিটার্ন করবে যাতে object এর property গুলোর value গুলো থাকবে।

একইভাবে কোনো object এর property গুলোর value কে জোড়ায় জোড়ায় পেতে চাই তাহলে,
const pair = Object.entries(objectName);
এটি একটি array এর মধ্যে প্রত্যেক জোড়াকে array আকারে রিটার্ন করবে।

object এর কোনো property কে delete করতে চাইলে,
delete objectName.propertyName;
এটাকে আমরা একটু অন্য ভাবে ও করতে পারি। আর সেটি হল ঐ property কে বাদ দিয়ে নতুন একটি object তৈরি করা
const {deletededPropertyName, ...newObject} = objectName;

freeze করলে ঐ object কে আর কোনো ধরণের পরিবর্তন করা যায় না। শুধু read করা যায়।

Class:

class হল একটি ছাচ বা কাঠামো (template or blueprint) যার সাহায্যে কোনো একটি নির্দিষ্ট প্রকারের একটি অবজেক্ট তৈরি করা যায়। class এর নাম সবসময় বড় অক্ষরে শুরু করতে হয়। class এর মধ্যে variable, function সবই থাকতে পারে, তবে এখানে dicleration key অর্থাৎ var, let, const, function এই keyword গুলো থাকবে না। ধরা যাক, আমরা একটি product class বানাবো। এটা আমরা এভাবে করতে পারি:
class Product{
constructor(name, sku, price ){
this.name = name,
this.sku = sku,
this.price= price
}
}
const tupi = new Product("koli tupi", "kkd455", 250);
console.log(tupi)

অনেক সময় একটি common class থাকে, যেটা দিয়ে কোনো কিছুর common বৈশিষ্টগুলোকে ডিফাইন করা হয়। তারপর আলাদা আলাদা বৈশিষ্টের জন্য আলাদা আলাদা নতুন class তৈরি করা হয়। আর সেই class এর মধ্যে আগের class কে extends করা হয়। যেমন:
class Panjabi extends Product{

dot(.) notaition vs bracket ([]) notation:

Object এর property এর value access করতে হলে আমরা দুইভাবেই করতে পারি।

    Base-class (Prototype instance)
  1. with dot(.) notaition -> ObjectName.propertyName
  2. with bracket ([]) notation-> ObjectName['propertyName']
তবে যদি কোনো propertyName special charecter বা number দিয়ে শুরু হয় অথবা হাইফেন দিয়ে যুক্ত হয় তখন bracket ([]) notation ব্যবহার করা বাধ্যতামূলক। bracket ([]) notation এ property name একটি string হবে বা একটি variable এর মধ্যে হতে হবে।
